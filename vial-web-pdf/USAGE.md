# Vial Web PDF Exporter 使い方ガイド

このWebアプリケーションは、Vial互換キーボードに接続し、キーマップレイアウトをPDFファイルとしてエクスポートすることができます。

## 前提条件

- **ブラウザ**: WebHIDをサポートするブラウザ（**Google Chrome**、**Microsoft Edge**、**Opera**など）。
- **デバイス**: Vial互換ファームウェアを実行しているキーボード。

## 使い方

1.  **キーボードの接続**:
    -   **"Connect Keyboard"** ボタンをクリックします。
    -   ブラウザのポップアップが表示されます。リストからキーボードを選択し、**"接続"** をクリックします。
    -   *注意: キーボードが表示されない場合は、接続されているか、OSに認識されているかを確認してください。*

2.  **レイアウトの表示**:
    -   接続されると、アプリケーションは自動的にキーボードの定義と現在のキーマップを読み込みます。
    -   キーボードレイアウトが視覚的に表示されます。

3.  **レイヤーの選択**:
    -   **"Layer"** ドロップダウンメニューを使用して、異なるレイヤー（例：Layer 0、Layer 1など）を切り替え、キー割り当てを確認できます。

4.  **PDFのエクスポート**:
    -   **"Export PDF (All Layers)"** ボタンをクリックします。
    -   すべてのレイヤーのレイアウトを含むPDFファイルが生成され、コンピュータにダウンロードされます。

# Vial Web PDF Exporter Usage Guide

This web application allows you to connect to a Vial-compatible keyboard and export its keymap layout to a PDF file.

## Prerequisites

- **Browser**: A browser that supports WebHID, such as **Google Chrome**, **Microsoft Edge**, or **Opera**.
- **Device**: A keyboard running Vial-compatible firmware.

## How to Use

1.  **Connect Keyboard**:
    -   Click the **"Connect Keyboard"** button.
    -   A browser popup will appear. Select your keyboard from the list and click **"Connect"**.
    -   *Note: If your keyboard does not appear, ensure it is plugged in and recognized by your OS.*

2.  **View Layout**:
    -   Once connected, the application will automatically load your keyboard's definition and current keymap.
    -   You will see a visual representation of your keyboard layout.

3.  **Select Layer**:
    -   Use the **"Layer"** dropdown menu to switch between different layers (e.g., Layer 0, Layer 1, etc.) and view their key bindings.

4.  **Export PDF**:
    -   Click the **"Export PDF (All Layers)"** button.
    -   A PDF file containing the layout for all layers will be generated and downloaded to your computer.
